# 🏭 Smart Factory Maintenance System

A Python OOP project that evaluates the maintenance condition of industrial machines based on their operating hours and maintenance interval.

## Features

* Calculate machine usage ratio
* Evaluate machine health
* Determine maintenance status
* Generate a maintenance report

## Project Structure


machine.py
maintenance_evaluator.py
main.py
